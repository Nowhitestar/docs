### Table Description
The Ethereum.transaction_logs_decoded table contains decoded logs from Ethereum transactions, providing a human-readable view of the events emitted by smart contracts. Logs are essential for understanding the actions and state changes that occur during the execution of smart contracts. By decoding these logs, the table offers a more accessible and detailed perspective on blockchain activities.
